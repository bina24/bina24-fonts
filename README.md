# Bina24 Font

Combined [DejaVu Sans](https://github.com/web-fonts/dejavu-sans) and [BPG Web 001 Caps](https://github.com/web-fonts/bpg-web-001-caps) fonts.

### Install

```
npm install bina24/bina24-fonts --save
```
